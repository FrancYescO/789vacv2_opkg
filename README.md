Mirror of (some) OpenWRT Packages repo for Technicolor 789vac XTREAM 35B
Mirrored from https://www.ilpuntotecnico.com/forum/index.php/topic,78382.msg260105.html#msg260105

Edit your `/etc/opkg/customfeeds.conf` to include these lines:

    src/gz packages https://raw.githubusercontent.com/FrancYescO/789vacv2_opkg/xtream35b/packages
    
 Then run `opkg update`
 
 ## Most of ackages are untested!
